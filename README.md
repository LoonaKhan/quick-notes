## To Run
- git clone the repo and cd into the src directory
```sh
git clone https://github.com/MonEmperor/quick-notes/edit/main/README.md
cd quick-notes/src/
```
- install all packages via npm
```sh
npm install
```
- run the server
```sh
npm start
```
